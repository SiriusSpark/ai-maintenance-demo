# AI Maintenance Alarm System (Demo)

This is a small demo project for practicing AI-driven development workflow.

## Tech Stack

Frontend:
- React
- TypeScript
- Vite

Backend:
- Node.js
- Express

## Features

- Alarm list page
- Filter by equipment name and date
- Pagination
- Alarm detail page
- Update alarm status
- Upload evidence files

## Project Structure

ai-maintenance-demo
├ frontend (React)
└ backend (Node.js)

## Run Frontend

cd frontend
npm install
npm run dev

## Run Backend

cd backend
node index.js
